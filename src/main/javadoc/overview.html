Sketching Core Library

Overview

The Sketching Core Library provides hashing, sketching and off-heap memory access functionality from Java. Click on the package links above for more information and code examples.

This library is divided into four packages that constitute four distinct groups of functionality:

  • Hash
  • Theta
  • Memory
  • HLL
Note: If the requirements or promises of any method's contract are not fulfilled (that is, if there is a bug in either the method or its caller), then an unchecked exception will be thrown. The precise type of such an unchecked exception does not form part of any method's contract.
